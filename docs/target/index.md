# Learning Targets

## Beginner

After completing the beginner route, you can:

- Explain Pods, Deployments and Services.
- Create resources with imperative commands and YAML.
- Inspect events, logs and resource status.
- Scale, update and roll back applications.
- Diagnose basic image, selector and container failures.

## Intermediate

- Deploy frontend, API and database workloads.
- Use ConfigMaps, Secrets and persistent storage.
- Configure Ingress, probes, resource requests and limits.
- Apply namespaces, service accounts and RBAC.
- Diagnose DNS, networking and storage failures.

## Production

- Operate a cluster with safe rollout and rollback procedures.
- Apply observability, backup and security controls.
- Package applications with Helm.
- Automate validation and delivery through CI/CD.
